\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Urban\+Map$<$ T $>$}
Represents an urban map as a graph structure. 

This class models locations as vertices and roads as edges, supporting various operations such as adding and removing locations and roads, and managing connectivity.
