void Menu::chooseOption ( )
private

Displays a menu and prompts the user to choose an option.

This function displays a list of options for the user to choose from, handles the user input, and invokes the appropriate functions based on the selected option. It validates the user's input, ensuring that only valid numbers (1, 2, 3, or 0) are accepted. The menu continues to loop until the user selects the exit option (0).
